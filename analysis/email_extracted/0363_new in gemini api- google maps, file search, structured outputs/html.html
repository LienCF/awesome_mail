Build with rich geospatial data from Google Maps

Now you can use the Google Maps tool in the Gemini API to ground your apps in rich geospatial data and build products that can answer questions about local places, like opening hours, ratings, and more.


Combine the Gemini model’s reasoning capabilities with rich Google Maps data from over 250 million places.

Ground Gemini with your data, simply and affordably

Explore File Search, a fully managed RAG system built directly into the Gemini API, to ground Gemini with your data and deliver responses that are more accurate, relevant, and verifiable for your specific needs.

 

File Search storage and embedding generation at query time is free of charge. You only pay for creating embeddings when you first index your files, at a fixed rate. 
Try in the demo app
Read the docs
 

Get improved Structured Outputs

Structured Outputs in the Gemini API have expanded support for standard JSON Schema. This adds native compatibility for popular libraries like Pydantic and Zod, introduces support for anyOf (unions) and $ref (recursive schemas), and preserves the implicit field ordering of your schema.

 

Use these improvements for data extraction, populating databases or agent communication.
