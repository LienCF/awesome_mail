Termius New October Edition

⚙️ Restore your whole setup with Workspaces
Workspaces can now save and restore your entire working state, including working path, running commands, and your Local Terminal. No more changing directories or restarting tools.

How it works
  • Open your project folder, tail logs, run top, start vim—then save it as a Workspace template.
  • Next time, launch the workspace, and everything will come back exactly as you saved it.

🤖 Vibecoding on iOS, upgraded
Hook up your favorite CLI code agent to a host in its settings. Termius will update the host icon for quick recognition and add a ready-to-run snippet to start your agent instantly.

When connected, you can now talk to your agent right from the terminal. The new input field makes prompts easy to edit and supports speech-to-text, so you can literally ask your agent to handle tasks while you stay hands-on with the shell.
