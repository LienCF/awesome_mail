<b>NVIDIA GPU Operator v25.10.0 Is Now Available
</b>
<br> <br>




The NVIDIA GPU Operator is a Kubernetes operator that simplifies the deployment and management of all NVIDIA software required to provision GPUs in a Kubernetes cluster. NVIDIA GPU Operator v25.10.0 is the first version that is “Government Ready”, making it available for regulated industries where STIG compliance and FIPS support are required.&nbsp;




<br><br>
The NVIDIA GPU Operator now uses distroless images to reduce security vulnerabilities and CDI by default in the Container Toolkit. This release also provides support for the latest Blackwell platforms: HGX B300 and GB300 NVL72.

<b>The NVIDIA NeMo Data Designer Microservice Is Now Available
</b>
<br><br>
NVIDIA NeMo Microservices is a modular suite of tools for managing the AI agent lifecycle. It includes software for data processing, model fine-tuning and evaluation, reinforcement learning, policy enforcement, and system observability.&nbsp;
<br><br>
The new NVIDIA NeMo Data Designer Microservice provides domain experts a way to generate high-quality, task-specific synthetic data at scale through prompting or providing seed examples, configurable schemas and AI-powered generation models, for specializing enterprise AI agents.
