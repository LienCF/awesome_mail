Boltz 2 Is Available as a Downloadable NIM

Boltz 2 is a structural biology foundation model that predicts complex biomolecular structures and quantifies molecular interactions. It includes support for protein structures, nucleic acid structures, protein-nucleic acid complexes, and ligand binding and affinity prediction.

The model supports both single molecule predictions and complex multi-molecular assemblies, making it suitable for a wide range of research applications from basic structural biology to drug discovery.

NVIDIA GPU Operator v25.3.4 Is Now Available

NVIDIA GPU Operator automates the lifecycle management of the software required to expose GPUs on Kubernetes. It enables advanced functionality including better GPU performance, utilization, and telemetry. Certified and validated for compatibility with industry-leading Kubernetes solutions, GPU Operator allows organizations to focus on building applications, rather than managing Kubernetes infrastructure.

This release adds comprehensive support for the latest NVIDIA hardware platforms: NVIDIA RTX PRO 6000 Blackwell Server Edition and NVIDIA RTX 6000D.

New capabilities include:
  • Support for MIG Profiles
  • Support for KubeVirt/OpenShift Virtualization, enabling VMs to attach to GPUs via passthrough or vGPU devices (requires vGPU v19)
