🚀 Universe highlights: what to try first
----------------------------------------

( https://github.com/events/universe/recap )

Universe 2025 focused on agents and getting from idea to shipping—faster. Start with these four updates:

* *Agent HQ* — Mission control for coding agents across GitHub and VS Code. Pick the best agent per task and keep your context in one place. Meet Agent HQ ( https://github.blog/news-insights/company-news/welcome-home-agents/ )
* *Agentic code review* — Copilot blends LLM reasoning with tools like CodeQL for deeper PR feedback. What’s new ( https://github.blog/changelog/2025-10-28-new-public-preview-features-in-copilot-code-review-ai-reviews-that-see-the-full-picture/ )
* *Plan Mode in VS Code* — Turn a prompt into a step-by-step plan before writing code—great for labs and projects. See Plan Mode ( https://github.blog/changelog/2025-10-28-github-copilot-in-visual-studio-code-gets-upgraded/ )
* *MCP integration in VS Code* — Safely discover/install Model Context Protocol servers to give Copilot new tools. Find MCP servers ( https://github.blog/ai-and-ml/github-copilot/meet-the-github-mcp-registry-the-fastest-way-to-discover-mcp-servers/ )

📊 Octoverse 2025: trends to shape your next project
---------------------------------------------------

( https://octoverse.github.com/ )

Octoverse is GitHub’s annual look at how the world codes—what languages are growing, how AI is used in real workflows, and where new contributors are getting started. Use it to decide what to learn next and how to talk about your work on resumes and in interviews.

* *Language momentum:* TypeScript leads the pack while Python and JavaScript remain core skills across student projects, internships, and open source.
* *AI in the dev loop:* AI-assisted tasks (tests, refactors, docs, code exploration) continue to grow—use Copilot to accelerate the “small wins” that unblock bigger features.
* *Onboarding to open source:* First contributions often start with documentation updates, good first issues, and tooling fixes—great entry paths for class or club projects.
* *Project hygiene matters:* Teams that automate checks (linting, CI, dependency updates) ship more confidently; bring these habits to your next assignment or hackathon.

*Pro tip:* Pull 1–2 charts that match your stack and turn them into a bullet on your resume (e.g., “Built a TypeScript + CI pipeline aligned with 2025 Octoverse trends”).

Read the full report ( https://octoverse.github.com/ )

🌍 Hack Club: make something big (ages 13–18)
--------------------------------------------

( https://hackclub.com/ )

Whether you’re into games or hardware, Hack Club has free, global programs for high-school students. Pick one and ship:

• *Moonshot* (Dec 12–15, Orlando): Ship projects, then fly to a 4-day hackathon with visits to Universal Studios and NASA Kennedy Space Center. Apply ( https://moonshot.hackclub.com/ )
• *Blueprint* : Design/document a hardware project and apply for up to $400 in parts. Start ( https://blueprint.hackclub.com/ )
• *Midnight* (Vienna, Jan): The world’s first murder-mystery hackathon—travel grants available. RSVP ( https://midnight.hackclub.com/ )
• *Milkyway* : Make games, build worlds, and compete for prizes—with a chance to fly to a hyper-immersive game jam in Singapore. Join ( https://milkyway.hackclub.com/ )

Explore Hack Club ( https://hackclub.com/ )
