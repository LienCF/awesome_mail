1. Cline CLI (Preview) - An interactive, terminal-native agent + scriptable infrastructure

Install: npm install -g cline

The CLI isn't just "Cline in your terminal." It's Cline as a building block you can script, automate, and embed anywhere, or simply chat with, just like in your IDE.
