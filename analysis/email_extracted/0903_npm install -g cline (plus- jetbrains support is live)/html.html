Two big updates: Cline goes scriptable and runs in JetBrains

Hello!

 

We've been busy lately, and I'm excited to share two updates that make Cline work everywhere you do:

  1. Cline CLI (Preview) - Turn Cline into scriptable infrastructure
  2. JetBrains support - Native Cline in IntelliJ, PyCharm, WebStorm, and more

Both are powered by Cline Core, the same agent loop we extracted to make Cline truly platform agnostic.

 

1. Cline CLI (Preview) - An interactive, terminal-native agent + scriptable infrastructure

Install: npm install -g cline

The CLI isn't just "Cline in your terminal." It's Cline as a building block you can script, automate, and embed anywhere, or simply chat with, just like in your IDE.

 

Our wizard Andrei, who led the work on extracting Cline Core, shared a great deep dive on its architecture and what it unlocks: https://cline.bot/blog/cline-cli-my-undying-love-of-cline-core

2. Cline is native in JetBrains

If you're a JetBrains user, Cline now runs natively in your IDE with the same Plan & Act modes and settings you know from VS Code. Install Cline for JetBrains: https://cline.bot/jetbrains

 

Try them out

The CLI is in preview - we're actively refining based on your feedback. Join #cli in our Discord (https://discord.gg/cline) to share what you're building, and report issues on GitHub.

 

Both updates are available now. Let us know what you think!
