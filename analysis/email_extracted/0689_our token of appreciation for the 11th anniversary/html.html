AI-powered Auto-Categorization in Microsoft Photos

Auto-Categorization automatically detects and organizes your photo collection into meaningful categories like screenshots, receipts, identity documents, and notes using AI.

This feature is available to Insiders in all channels who are using Copilot+ PCs.

Launch network speed tests directly from your taskbar

 

Right-click the network icon in the system tray. The tool supports testing Ethernet, Wi-Fi, and Cellular connections.


This feature is rolling out to Insiders in the Dev Channel and Beta Channel.

Copilot-powered translations in Click to Do

Translate on-screen text with just a few clicks! When you select text that is in a different language from your Windows display language or preferred language settings, a translation suggestion will appear.

Currently available to Insiders in the Dev Channel and Beta Channel on Copilot+ PCs outside the EEA (European Economic Area) and China.
Learn more about Copilot-powered translations

Windows 11 Tips and Tricks

 

Did you know you can mark favorite files in File Explorer?

Just right click the file in File Explorer and select Add to Favorites – then it will be listed in the Favorites section of Home for easy access.
Learn more about File Explorer  ❯

Want quick access to Device Manager?

If you right click the Start menu icon in the taskbar, a menu will open with entries for Device Manager, shutdown/restart, Task Manager, and more!
Learn about more Start menu options  ❯

One of the benefits to ordering the apps in the taskbar…

Is that you can launch them by pressing the Windows key plus the number corresponding to their position.
Learn more keyboard shortcuts  ❯
