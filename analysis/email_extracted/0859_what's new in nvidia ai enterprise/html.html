NVIDIA Riva Canary 1B ASR/AST and NVIDIA Riva Parakeet Taiwanese Mandarin Are Now Available as Downloadable NIM

NVIDIA Riva Canary 1B ASR/AST is a multimodal model for speech to text recognition (ASR) in 26 languages and speech to text translation (AST) in 34 languages. Trained using NVIDIA’s Granary dataset using a FastConformer Encoder and a Transformer Decoder, the model supports .wav and .flac audio files for input, and outputs text, including automatic punctuation and capitalization. The new model is offline ready with In-Flight Batch Mode (IBD) and has improved accuracy and throughput. It replaces Canary 1B and Canary 0.6 Turbo, which are now deprecated.

NVIDIA Riva Parakeet 0.6B Taiwanese Mandarin is trained on an ASR dataset with around 90 hours of Taiwanese Mandarin (zh-TW) speech. 

NVIDIA Riva Magpie TTS Multilingual NIM has added Mandarin, in addition to English, Spanish, French and German, and now supports up to 64 concurrent streams via REST and WebSocket APIs. 

All three NIM are available for commercial use with a NVIDIA AI Enterprise license.

3 NIM Optimized for NVIDIA DGX Spark Are Now Available

NVIDIA Nemotron Nano 9B v2, Meta Llama 3.1 8B and Alibaba Qwen 3 32B are available as downloadable NIM optimized for NVIDIA DGX Spark. 

DGX Spark systems, built on the NVIDIA GB10 Grace Blackwell Superchip, deliver up to 1 petaflop of AI performance in a compact desktop form factor, giving developers the power to run inference on AI models with up to 200B parameters and fine-tune models of up to 70B parameters locally. The systems also feature 128GB of CPU-GPU coherent memory and NVIDIA NVLink-C2C technology for high-speed connection.

The NVIDIA AI software stack is preinstalled on DGX Spark to enable developers to start working on AI projects out of the box.
