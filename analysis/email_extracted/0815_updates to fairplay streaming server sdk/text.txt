Hello,

We’re reaching out to let you know that FairPlay Streaming Server SDK 26 is now available. This release includes important security enhancements and interface updates, and we strongly recommend updating your deployment to maintain the highest level of protection for your content.

What you need to know:

• Effective immediately, you can only generate new certificates for SDK 26 and SDK 4.

• Starting September 2026, Apple will no longer provide technical support for SDKs 4 and 5.

• Existing deployments will continue to function in the same way, and existing certificates will not expire.

• Update your deployment to benefit from the latest security enhancements and interface updates.
